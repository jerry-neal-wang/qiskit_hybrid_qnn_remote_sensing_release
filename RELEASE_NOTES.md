# Release Notes

## release_v2

Purpose: GitHub/review package corresponding to the v3 GRSL submission-ready paper.

Key updates from the earlier release package:

- Aligns the paper package with the final v3 LaTeX manuscript and supplement.
- Uses the current single-mainline codebase for DIOR low-shot hybrid QNN classification.
- Excludes removed project routes such as BB84, secure return, NEQR, route2, and legacy quantum-image experiments.
- Adds current lightweight tests and the unified environment file.
- Keeps only clean CSV evidence and metadata; raw run directories, checkpoints, and raw DIOR files are excluded.
- Updates the manuscript Data and Code Availability URL to the `release_v2` GitHub target.

The scientific claim remains bounded: the observed hybrid-QNN gain is limited to the reported low-label `transport_logistics4` small-CNN simulation regime and contracts under the submitted boundary checks.
